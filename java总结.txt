 java JVM包括：
 1.程序计数器
 2.本地方法栈
 3.虚拟机栈：描述java方法的内存模型，每一个方法调用到执行结束的过程都对应着一个栈帧在虚拟机栈从入栈到出栈的过程
 4.堆：存放对象实例，创建对象会在堆中开辟空间，垃圾收集器管理的主要区域，故又称"GC堆"
 5.方法区及其中的运行时常量池：存储静态变量虚拟机加载的类信息，常量，静态变量

类加载过程：
1.加载要创建对象的的类及其父类；
2.在类加载的同时会将静态成员进行加载，主要包括静态成员变量的初始化，静态代码块的执行；
3.需要的类加载完成后，开始创建对象，首先会加载非静态成员变量，非静态代码块的执行；
4.执行构造器，执行完毕，对象生成。
